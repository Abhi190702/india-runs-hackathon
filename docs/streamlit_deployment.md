# Streamlit Deployment

1. Push the repository to GitHub.
2. Open Streamlit Cloud.
3. Create a new app.
4. Select the repository.
5. Set the main file to `app/streamlit_app.py`.
6. Deploy.
7. Add the deployed link to `README.md` and `submission_metadata.yaml`.

The app is a demo sandbox. It uses bundled sample candidates or uploaded JSON/JSONL samples and does not require the full organizer dataset.
